À propos

Système de Gestion de Matériel

Le Système de Gestion de Matériel (SGM) est une application conçue pour simplifier la gestion des ressources informatiques d'une organisation. Notre objectif est de fournir une solution simple et efficace pour suivre l'inventaire, gérer les affectations et assurer la maintenance du matériel informatique.

Fonctionnalités

  • Suivi des stocks en temps réel
  • Gestion des affectations de matériel
  • Historique des mouvements
  • Alertes de stock bas
  • Rapports personnalisables
  • Interface intuitive et responsive

Technologies utilisées

  • Frontend : HTML, CSS, JavaScript
  • Backend : Python (Flask)
  • Logique métier : Lua
  • Base de données : SQLite (simulation)
